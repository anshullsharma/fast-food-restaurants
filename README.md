# 🍔 Fast Food Restaurant Web App

## 📌 Overview
A **Django-based** web application that allows users to browse a restaurant's menu, place orders, and track their status. It also provides an **admin panel** for managing orders, customers, and menu items.

## 🚀 Features
✅ **User Authentication** – Secure login & signup using Django authentication.  
✅ **Menu Display** – Users can browse the restaurant’s menu.  
✅ **Order Management** – Customers can add items to the cart, place orders, and track them.  
✅ **Admin Dashboard** – Restaurant owners can manage orders, inventory, and customers.  
✅ **Responsive UI** – Works seamlessly on all devices.
