# 🍔 Fast Food Restaurant Web App

## 📌 Overview
A **Django-based** web application that allows users to browse a restaurant's menu, place orders, and track their status. It also provides an admin panel for managing orders and customers.

## 🚀 Features
- User authentication (Sign up, login, logout)
- Browse menu and add items to cart
- Order tracking and history
- Admin dashboard for order and inventory management

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Django (Python)  
- **Database:** SQLite
